# 🍒 Cereja & Baunilha - Confeitaria Artesanal

Este é um projeto de Landing Page desenvolvido para a confeitaria **Cereja & Baunilha**. O objetivo foi criar uma experiência visual sofisticada, focada em converter visitantes em clientes através de um design elegante e moderno.

## 🚀 Funcionalidades
- **Design Responsivo:** Site adaptável para computadores, tablets e celulares.
- **Animações de Scroll:** Implementação da biblioteca AOS para efeitos de *fade-in* e surgimento suave de elementos.
- **Galeria de Produtos:** Portfólio visual dos doces utilizando efeitos de hover e zoom.
- **Integração Direta:** Botão de WhatsApp configurado para pedidos imediatos.
- **Prova Social:** Seção de depoimentos de clientes para gerar confiança.

## 🛠️ Tecnologias Utilizadas
- **HTML5**: Estruturação semântica.
- **CSS3**: Estilização avançada, Flexbox, CSS Grid e Variáveis de Cores.
- **JavaScript**: Lógica de interação e animações.
- **AOS Library**: Para animações de rolagem de página.
- **Google Fonts**: Tipografia selecionada (*Playfair Display* e *Montserrat*).
